Project Documents
Software Developers and Project Managers: You can help us outmaneuver COVID-19!

Problem: As COVID-19 spreads, it can mutate. Mutations make vaccines and tests ineffective.

Solution: Pantograph—a War Room Tool for biologists to track mutations. Real-time visualization empowers us to stay ahead of the fight with new vaccines, tools, and tests.

Method: Open Source project on GitHub. Features will be broken down into issues.

Goal: Open Source data collection point for:

  • Health care providers to note mutations on Pantograph
  • Biologists to immediately flag mutations on a global map and track new strains
  • Vaccine developers to stay ahead of mutations with new vaccines

What we need: Software Developers (no biology background necessary) and Project Managers. 10-20 hours/week volunteer time to develop this crucial tool. Most beneficial skills: JavaScript, React, Semantic Web using SPARQL

Software Developers:

  • You’ll work on an issue and will make a pull request
  • Each team’s Features will be merged into a larger project
  • We’ll create a powerful War Room Tool together

Project Managers:

  • You’ll work on a Project and will monitor task progress
  • You’ll manage communication between developers
  • You’ll oversee testing codes, finding bugs, and posting issues

Join us: Drop a mail to josiah@newline.us and get involved!

The Pantograph project is supported by

The deNBI Cloud has provided VMs for developing this project and is currently providing a VM of 28 cores, 64GB RAM and 1TB storage for our COVID19 Pangenome Visualization. The deNBI cloud covid-19-resources lists us as one of the available resources to fight COVID19.

Pangenome Schematic design and development research was funded through the 1001 Genomes project under grant BBSRC BB/S004661/1.

NBDC/DBCLS BioHackathon 2019 is supported by the Integrated Database Project (Ministry of Education, Culture, Sports, Science and Technology of Japan) and hosted by the National Bioscience Database Center (NBDC) and the Database Center for Life Science (DBCLS).

Computomics GmbH funded a second hackathon to implement the data pipeline in November 2019.

Toshiyuki Yokoyama’s work was funded in part by JSPS KAKENHI (grant numbers 16H06279 and 19J21608).
